# Deprecated — Old Netlist Export Files

These files were produced by the `firmware/tools/convert_netlist_to_json.py` script as part of the original annotated netlist system. That system has been replaced by the **schematic context system**.

**Use instead:** `hardware/pcb-main/kicad/circuit-context.json`

The circuit-context.json file is produced by `tools/schematic-context/extract.py` and contains everything these files provided plus semantic annotations, functional block groupings, and design intent notes.

These files are retained for reference but are no longer maintained or used.
