# Deprecated — Legacy Sensor PCB Netlist Files

These files are from the original combined sensor PCB design (`pcb-sensor`), which was split into separate `sensor-rigid` and `sensor-flex` designs.

The annotation YAML file (`*.annotations.yaml`) was part of the old annotated netlist system, which has been replaced by the **schematic context system** (`tools/schematic-context/`).

For current sensor PCB designs, see `hardware/sensor-rigid/` and `hardware/sensor-flex/`. Circuit-context.json files for these boards will be generated in a future pass using the same tooling.
